# 最优化算法可视化清单（配 Claude Design / Canva 实现）

> 按「可视化价值 × 考试权重」排序。每条给出：**画什么 → 为什么（考试关联）→ 可直接喂给 Claude Design 的中文 prompt → 关键标注 → 工具建议**。
>
> **分流原则**：
> - 概念示意图 / 信息图 / 对比图 → Claude Design (Canva) 最擅长。
> - ⚡ 标记的：精确函数曲线/等高线建议先用 **Desmos / GeoGebra** 画好截图，再拖进 Canva 加箭头和标注（比纯手画准）。
> - 统一配色建议：**最优点 = 红，迭代路径 = 绿/蓝，约束 = 灰**，跨图一致，形成肌肉记忆。

## 第一梯队（强烈推荐：几何直觉强 + 考试重点）

### ① 梯度下降"锯齿" + 条件数影响 ⚡
**对应**：模拟题三、[详细笔记/08_梯度类算法](详细笔记/08_梯度类算法.md)

- **画什么**：左右两张等高线对比。左=正圆等高线（κ=1），绿箭头一步直达中心；右=扁椭圆（κ大），红折线之字形横跳逼近中心。
- **为什么**：梯度法的灵魂直觉，解释"条件数/步长为何决定快慢"，纯文字最难想象。
- **Claude Design prompt**：
  > 画一张教学对比信息图，左右两个面板。左面板标题"条件数κ=1（理想）"，画一组同心圆等高线，中心一个红点x*，从外侧一个点用绿色直箭头一步指向中心。右面板标题"条件数κ大（病态）"，画一组水平方向拉长的扁椭圆等高线，从右上角的点画红色折线箭头做之字形锯齿状逐步靠近中心。底部大标注"等高线越扁 → 锯齿越严重 → 收敛越慢"。极简扁平风格，蓝绿配色。
- **关键标注**：相邻搜索方向正交 ⟹ 锯齿；收缩率 $\left(\frac{\kappa-1}{\kappa+1}\right)^2$。

### ② 牛顿法 vs 梯度法 路径对比
**对应**：[详细笔记/09_牛顿类算法](详细笔记/09_牛顿类算法.md)

- **画什么**：同一组扁椭圆等高线上叠加两条路径：梯度法红色锯齿折线（多步），牛顿法蓝色直线（1~2步直达）。
- **为什么**：一眼看懂"牛顿用二阶信息把椭圆掰圆，所以走直线"。
- **Claude Design prompt**：
  > 在一组扁椭圆等高线背景上，画两条从同一起点到中心x*的路径对比：一条红色虚线做多次锯齿折返并标注"梯度法 慢（线性收敛）"，一条蓝色实线几乎直达中心并标注"牛顿法 快（二次收敛）"。右侧加小注"牛顿法用Hessian把扁椭圆掰成圆"。简洁教学风。
- **关键标注**：二次型牛顿法一步到位 $x^*=Q^{-1}b$。

### ③ 驻点三类型：极小 / 极大 / 鞍点
**对应**：模拟题二、[详细笔记/04_无约束最优性条件](详细笔记/04_无约束最优性条件.md)

- **画什么**：三个并排 3D 曲面缩略图 + 对应等高线：碗（极小）、倒碗（极大）、马鞍（鞍点），各标 ∇f=0 与 Hessian 定号。
- **为什么**：模拟题二就是分类驻点，鞍点最难脑补。
- **Claude Design prompt**：
  > 画一张三联教学卡片图，并排三个卡片。卡片1"局部极小"：一个向上的碗形3D曲面+底部红点，标注"∇f=0, Hessian正定"。卡片2"局部极大"：倒扣的碗形曲面+顶部红点，标注"∇f=0, Hessian负定"。卡片3"鞍点"：马鞍形曲面+中间红点，标注"∇f=0, Hessian不定"。统一配色，每张卡片下方写判定条件。
- **关键标注**：配方法判全局；全局最小可能不唯一。

### ④ KKT / 拉格朗日几何："相切 + 梯度共线" ⚡
**对应**：模拟题四、[详细笔记/06_KKT条件](详细笔记/06_KKT条件.md)

- **画什么**：约束曲线（单位圆 $x_1^2+x_2^2=1$）+ 目标 $x_1x_2$ 的等高线（双曲线族），在最优点画 ∇f 与 ∇c 两箭头共线，标"相切处=最优"。
- **为什么**：KKT 最难理解的"为什么最优点梯度共线"，这图直接点破，且复用模拟题四例子。
- **工具**：⚡ 用 Desmos 画 `x^2+y^2=1` 和 `xy=k` 几条等高线，截图后导入 Canva 加箭头。
- **Claude Design prompt**（示意版）：
  > 画一个坐标系，中间一个灰色圆代表约束c(x)=0，圆上一点P标为红色最优点。在P点画两个箭头：一个标"∇f"、一个标"∇c"，两箭头在同一直线上（共线）。圆周附近画几条与圆在P相切的曲线代表目标等高线。大标注"最优点：目标等高线与约束相切，∇f = λ∇c"。
- **关键标注**：等式约束 $\nabla f=\lambda\nabla c$；解出全部候选后比较 f 值定 min/max。

### ⑤ 增广拉格朗日 ALM 收敛过程
**对应**：模拟题五、[详细笔记/12_罚函数法与增广拉格朗日法](详细笔记/12_罚函数法与增广拉格朗日法.md)

- **画什么**：左=乘子递推折线 $\lambda:0\to-1\to-1.5\to-1.75\to\cdots\to-2$；右=迭代点在直线 $x+y=2$ 上逐步逼近真解 (1,1)。
- **为什么**：直观展示"固定 σ、靠更新乘子也能精确收敛"这一核心优势。
- **Claude Design prompt**：
  > 画一张两栏信息图。左栏：一条折线图展示序列 λ: 0 → -1 → -1.5 → -1.75 → ... 用箭头逐步逼近一条虚线-2，标题"乘子λ收敛到-2"。右栏：一个坐标系画直线x+y=2，线上一串点逐步收敛到红点(1,1)，标题"迭代点逼近真解(1,1)"。底部标注"σ固定=1，靠更新乘子精确收敛"。
- **关键标注**：递推 $\lambda^{k+1}=\frac12\lambda^k-1$；极限 $\lambda^*=-2$，解 $(1,1)$。

## 第二梯队（锦上添花：概念更清晰）

### ⑥ 凸函数判定（弦 / 切线）⚡
**对应**：模拟题一、[详细笔记/02_凸函数](详细笔记/02_凸函数.md)
- **画什么**：一条上凸曲线 + 一条弦（在曲线上方）+ 一条切线（在曲线下方）。
- **prompt**：
  > 画一条碗形（凸）曲线，在曲线上取两点连一条直线段（弦），弦完全在曲线上方；再在曲线某点画一条切线，切线完全在曲线下方。标注"凸函数 = 曲线在弦下方 = 在切线上方"。
- **标注**：判据 $f''\ge0$ / Hessian 半正定。

### ⑦ L1 vs L2 稀疏性 ⚡
**对应**：[详细笔记/03_凸优化问题与稀疏优化](详细笔记/03_凸优化问题与稀疏优化.md)
- **画什么**：菱形(L1球)与圆(L2球)分别和一组椭圆等高线相切；L1 切在坐标轴尖角 → 出 0（稀疏），L2 切在普通点 → 不稀疏。
- **prompt**：
  > 左右两个坐标系。左图画一个菱形（L1球）和一组椭圆等高线相切，切点落在坐标轴的尖角上，标注"L1正则 → 解稀疏（出0）"。右图画一个圆（L2球）和椭圆等高线相切于一般点，标注"L2正则 → 解不稀疏（压系数）"。
- **标注**：LASSO 用 L1 选变量，岭回归用 L2 压系数。

### ⑧ 线搜索 Armijo / Wolfe 接受区间 ⚡
**对应**：[详细笔记/07_线搜索方法](详细笔记/07_线搜索方法.md)
- **画什么**：一维 $\phi(\alpha)=f(x+\alpha d)$ 曲线 + Armijo 参考直线，高亮"可接受步长区间"。
- **prompt**：
  > 画一条先降后升的一维曲线φ(α)，从原点画一条斜率较缓的下降直线（Armijo参考线），把曲线在参考线下方的α区间用绿色高亮，标注"Armijo可接受区间"。再标一个点表示Wolfe额外的曲率条件。
- **标注**：Armijo 够降；Wolfe 加曲率（拟牛顿必配）。

### ⑨ 共轭方向 vs 正交方向
**对应**：[详细笔记/11_共轭梯度法](详细笔记/11_共轭梯度法.md)
- **画什么**：左=圆里一对垂直直径（普通正交）；右=椭圆里一对共轭直径。
- **prompt**：
  > 左图一个圆，画两条互相垂直的直径，标"普通正交 uᵀv=0"。右图一个椭圆，画一对共轭直径（其中一条的端点切线平行于另一条），标"共轭 uᵀAv=0（带权重A的正交）"。箭头标注"圆拉伸成椭圆，垂直变共轭"。
- **标注**：共轭方向各走一次 → n 步收敛。

### ⑩ 收敛速率对比（半对数图）⚡
**对应**：[详细笔记/07_线搜索方法](详细笔记/07_线搜索方法.md)
- **画什么**：同一张半对数（y轴 log 误差）图三条曲线：线性=直线下降、超线性=渐陡、二次=极陡断崖。
- **prompt**：
  > 画一张半对数坐标图（纵轴为误差的对数），三条曲线：一条直线下降标"线性收敛（梯度法）"，一条逐渐变陡标"超线性（BFGS）"，一条几乎垂直断崖式下降标"二次收敛（牛顿法）"。标注"越陡越快"。
- **标注**：复杂度 线性 $O(\log\frac1\varepsilon)$ < 二次 $O(\log\log\frac1\varepsilon)$。

## 实操建议

1. **优先做 ①③④** —— 几何直觉 × 考试权重双高，理解会质变。
2. **④⑥⑦⑧⑩（带 ⚡）** 需要精确曲线的，先在 **Desmos**（免费网页版，输函数即出图）画好截图，再拖进 Canva 加箭头文字，比纯手画准得多。
3. prompt 里都强调"**标注结论文字**"——可视化一半价值在图、一半在那句点睛标注，别省。
4. 跨图统一配色（最优点红 / 路径绿蓝 / 约束灰），复习时形成视觉记忆。
