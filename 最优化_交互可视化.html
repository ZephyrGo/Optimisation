① 固定步长梯度下降的收敛性 模拟题三-1

目标 f(x)=½x²，迭代 xk+1=(1−α)xk。拖动步长 α，看序列怎么变。

收敛 ⟺ |1−α|<1 ⟺ 0<α<2；
α=1 收缩因子为0 → 一步到底（最快，=1/L）；
α=2 → 因子−1 → 两点震荡；α>2 → 发散。

② 下降算法对比 + 条件数 + 锯齿 模拟题三 / 笔记08-09

二次型 f=½(λ₁x²+λ₂y²)。在画布上点击设起点，选算法后点"运行"，看迭代路径在椭圆等高线上的走法。

点击画布设置起点…
条件数 κ=λ₁/λ₂。κ越大椭圆越扁，梯度法锯齿越狠越慢；牛顿法用Hessian把椭圆"掰圆"，一步直达。

③ KKT / 拉格朗日"相切+梯度共线" 模拟题四

min f=x₁x₂ s.t. x₁²+x₂²=1。拖动 θ 让点 P 沿约束圆走，观察 ∇f(蓝) 与 ∇c(红) 何时共线——那就是 KKT 点。

共线条件 ∇f=λ∇c。本例 KKT 点在 θ=45°,135°,225°,315°：
f=−½（min，θ=135°/315°）或 f=+½（max，θ=45°/225°）。
灰双曲线是目标 x₁x₂ 的等高线，相切处即最优。

④ 增广拉格朗日法 ALM 收敛 模拟题五

min x²+y² s.t. x+y−2=0，σ=1。点"下一步"看乘子 λ 递推 0→−2、迭代点逼近真解 (1,1)。

子问题极小：x=y=(2−λ)/4；
乘子递推：λk+1=λk+σ·c = ½λk−1 → 极限 λ*=−2；
代回 x=y=1。σ固定=1（不必→∞），靠更新乘子精确收敛。

⑤ 驻点类型：极小 / 极大 / 鞍点 模拟题二 / 笔记04

热力图（蓝=低，红=高）。切换函数，观察驻点(★)处的形态。∇f=0 处看 Hessian 定号判类型。

判定：∇f=0 求驻点 → Hessian 正定=极小 / 负定=极大 / 不定=鞍点。
模拟题二函数有 3 个驻点：(0,0)鞍点，(±1,0)严格全局极小(f=−1)。
