# 02 凸函数

> 对应课件 1-3。凸函数是我们真正要优化的对象。本章给你三套"判断凸不凸"的工具，以及它们各自适合什么场合。

## 0. 为什么凸函数这么重要

一句话：**凸函数没有"假最优点"**。

非凸函数像连绵起伏的山脉，你走到一个山谷（局部最低），但翻过去可能还有更深的谷，你却不知道。凸函数像一个**碗**，只有一个最低点，你顺着往下走，到底了就是全局最优，不会被骗。

所以"判断一个函数是不是凸的"是一项基本功。

---

## 1. 定义：碗形

### 定义
函数 $f$ 是**凸函数**，如果它的定义域是凸集，且对任意两点 $x,y$：

$$f(\theta x+(1-\theta)y)\le \theta f(x)+(1-\theta)f(y),\quad \forall\theta\in[0,1]$$

### 大白话 + 几何
左边是"两点中间某处的函数值"，右边是"两点函数值的连线（弦）在该处的高度"。

不等式说：**函数曲线永远在它的弦（割线）下方**。

```
   f(x)·              ·f(y)
       \            /     ← 这条直线是"弦"
        \          /
         \___..___/        ← 函数曲线在弦下方 = 凸
```

> 反过来，**凹函数**是曲线在弦上方（碗倒扣，山峰形）。$f$ 凸 ⟺ $-f$ 凹。求 $\max$ 凹函数 = 求 $\min$ 凸函数，本质一回事。

### 上图（epigraph）—— 把函数变集合
$f$ 的**上图**是"函数曲线及其上方的所有点"：$\text{epi}f=\{(x,t):f(x)\le t\}$。

**关键事实**：$f$ 是凸函数 $\iff$ epi$f$ 是凸集。

这把[上一章](01_凸集.md)的凸集理论直接搬到了函数上 —— 很多关于凸函数的结论，本质就是 epi$f$ 这个凸集的性质。

---

## 2. 三套判定工具（重点！）

实际做题，按手头有什么信息选工具。

### 工具一：定义法（零阶，最通用但最难用）
直接验证 $f(\theta x+(1-\theta)y)\le\theta f(x)+(1-\theta)f(y)$。
- 优点：不需要可导。
- 缺点：通常很难硬算。一般只在函数很简单、或证明题里用。

### 工具二：一阶条件（需要可导）

$$f(y)\ge f(x)+\nabla f(x)^T(y-x)\quad \forall x,y$$

**几何**：右边是 $f$ 在 $x$ 点的**切线（切平面）**。不等式说：**函数永远在它任意一点的切线上方**。

```
        f
       /
      /·------ 切线
     /
    ·  ← 切点 x
   函数全程在切线之上 = 凸
```

**这个条件极其重要**，因为它直接通向最优性：
- 如果某点 $x^*$ 处 $\nabla f(x^*)=0$，代入得 $f(y)\ge f(x^*)$ 对所有 $y$ 成立，即 $x^*$ 是**全局**最小点！
- 这就是"凸函数里，梯度为零 = 全局最优"的来源（[04章](04_无约束最优性条件.md)细讲）。

### 工具三：二阶条件（需要二阶可导，最常用）

$$\nabla^2 f(x)\succeq0\quad \forall x \quad(\text{Hesse 矩阵半正定})$$

**一元情形**就是 $f''(x)\ge0$（二阶导非负 = 曲线向上弯 = 凸）。

**多元情形**：Hesse 矩阵半正定。怎么判半正定？
- 所有特征值 $\ge0$；或
- 所有顺序主子式 $\ge0$；或
- 对任意 $v$ 有 $v^T\nabla^2 f(x)v\ge0$。

> **这是实战中最常用的判据**。给你一个具体函数，求二阶导/Hesse，看定不定号，搞定。

**例**：$f(x)=\frac12 x^TQx-b^Tx$，$\nabla^2 f=Q$。所以这个二次函数凸 $\iff Q\succeq0$，严格凸 $\iff Q\succ0$。后面算法天天用到。

## 3. 强凸与 L-光滑：给凸性"定量"

光说"凸"还不够，算法收敛快不快，取决于"凸得多厉害"和"弯得多狠"。引入两个量化指标。

### 3.1 强凸（凸得有下限）
$f$ 是 **$m$-强凸**，如果 $\nabla^2 f(x)\succeq mI$（$m>0$），等价于 $f(x)-\frac{m}{2}\|x\|^2$ 还是凸的。

**直觉**：碗底不能太平，至少要有 $m$ 这么陡。强凸 ⟹ 最优解**唯一**，且算法收敛更快（线性收敛）。

### 3.2 L-光滑（弯得有上限，= 梯度 Lipschitz 连续）
$f$ 是 **$L$-光滑**，如果 $\|\nabla f(x)-\nabla f(y)\|\le L\|x-y\|$，等价于 $\nabla^2 f(x)\preceq LI$。

**直觉**：梯度变化不能太剧烈，曲率有上界 $L$。

**最有用的推论 —— 二次上界**：

$$f(y)\le f(x)+\nabla f(x)^T(y-x)+\frac{L}{2}\|y-x\|^2$$

意思是：L-光滑函数可以被一个**二次函数从上面盖住**。这是几乎所有梯度类算法收敛性证明的出发点（[08章](08_梯度类算法.md)会反复用）。

### 把两者合起来看
如果 $mI\preceq\nabla^2 f\preceq LI$，那么 $f$ 被夹在两个二次函数之间。比值 $\kappa=L/m$ 叫**条件数**，它决定了梯度法快慢：$\kappa$ 越接近 1 越快，$\kappa$ 很大（病态）就很慢。这个 $\kappa$ 会贯穿后面所有算法章节。

---

## 4. Jensen 不等式（凸性的"应用神器"）

凸函数定义推广到多个点 / 概率：

$$f\!\left(\sum\theta_i x_i\right)\le\sum\theta_i f(x_i),\qquad f(\mathbb{E}X)\le\mathbb{E}f(X)$$

**用处**：很多经典不等式（均值不等式、信息论里的熵不等式）都是 Jensen 的特例。[极值应用案例](../最优化_重点与例题.md)里用它推出"调和≤几何≤算术≤平方"均值链。

---

## 5. 保凸运算：怎么"造"凸函数

和凸集一样，实战靠组合已知凸函数，而不是死算。记住这几条：

| 运算 | 说明 | 例子 |
|---|---|---|
| **非负加权和** | $\sum w_if_i$（$w_i\ge0$）凸 | $\|Ax-b\|^2+\mu\|x\|_1$ 两个凸项加起来还是凸 |
| **与仿射复合** | $f(Ax+b)$ 凸 | 把 $\|x\|$ 变成 $\|Ax-b\|$ 仍凸 |
| **逐点最大/上确界** | $\max_i f_i(x)$ 凸 | 一堆线性函数取 max 得到分段线性凸函数 |
| **部分极小** | 对部分变量取 min（需联合凸） | 推导对偶时用 |

> **注意方向**：逐点 **max** 保凸（max 让函数更"尖"，碗更深），逐点 **min** 一般**不**保凸。别记反。

---

## 小结与下一步

- 凸函数 = 曲线在弦下方 = 切线上方 = epi 是凸集 = Hesse 半正定（多套等价说法，按手头信息挑工具）。
- 实战首选**二阶条件**（看 Hesse 定号）。
- **一阶条件**是通往"梯度为零即全局最优"的桥。
- 强凸（$\succeq mI$）和 L-光滑（$\preceq LI$）给凸性定量，条件数 $\kappa=L/m$ 决定算法快慢。
- 造凸函数靠保凸运算（非负和、仿射复合、逐点 max）。
