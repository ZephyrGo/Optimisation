# 07 线搜索方法

> 对应课件 2-1。从这章起进入"算法"。第一章告诉我们最优点长什么样，现在要造一台"机器"，从随便一个起点出发，一步步走到那里。

## 0. 迭代算法的总框架

几乎所有无约束优化算法都长这个样子：

$$x^{k+1}=x^k+\alpha_k d^k$$

- $x^k$：当前位置（第 $k$ 步）。
- $d^k$：**搜索方向**（往哪个方向走）。
- $\alpha_k$：**步长**（这个方向走多远）。

不同算法的区别，全在**怎么选方向 $d^k$** 和**怎么选步长 $\alpha_k$**：
- 方向怎么选 → [梯度法](08_梯度类算法.md)（负梯度）、[牛顿法](09_牛顿类算法.md)（牛顿方向）、[拟牛顿](10_拟牛顿类算法.md)、[共轭梯度](11_共轭梯度法.md)……
- **步长怎么选 → 就是本章"线搜索"的任务**。

所以本章是所有算法共用的"步长引擎"，先学它。

---

## 1. 先搞懂"收敛速率"：算法快慢怎么衡量

在比较算法前，得有个统一的"快慢标尺"。设 $e_k=\|x^k-x^*\|$ 是第 $k$ 步离最优点的误差，看 $e_k$ 怎么往 0 缩：

| 名称 | 关系 | 快慢 | 谁是代表 |
|---|---|---|---|
| **次线性 sublinear** | $e_k\sim O(1/k)$ 这类 | 最慢 | 一般凸问题的梯度法 |
| **Q-线性 linear** | $e_{k+1}\le c\,e_k$（$0<c<1$） | 中等（误差等比缩小） | 强凸的梯度法 |
| **Q-超线性 superlinear** | $e_{k+1}/e_k\to0$ | 快 | 拟牛顿（BFGS） |
| **Q-二次 quadratic** | $e_{k+1}\le c\,e_k^2$ | 最快（误差位数翻倍） | 牛顿法 |

### 直观感受"翻倍"有多猛
二次收敛下，误差大致 $0.1\to0.01\to0.0001\to10^{-8}$，**每步有效位数翻倍**，几步就到机器精度。线性收敛则是 $0.1\to0.05\to0.025\dots$ 慢慢减半。这就是牛顿法"几步搞定"而梯度法"磨蹭半天"的根源。

### 复杂度 $N(\varepsilon)$
达到精度 $\varepsilon$ 需要多少步。比如次线性 $O(1/\varepsilon)$、线性 $O(\log\frac1\varepsilon)$、二次 $O(\log\log\frac1\varepsilon)$。$\log\log$ 几乎是常数，再次说明二次收敛之快。

> 但注意：牛顿法虽然步数少，**每步更贵**（要算 Hesse、解方程组）。快慢要看"步数 × 每步代价"，这是后面选算法的关键权衡。

## 2. 精确线搜索：把步长这一维问题解到底

### 思路
方向 $d^k$ 定好后，沿这条线 $f(x^k+\alpha d^k)$ 只是关于 $\alpha$ 的**一元函数** $\phi(\alpha)$。精确线搜索就是把它的最小值点真正求出来：

$$\alpha_k=\arg\min_{\alpha>0}\ \phi(\alpha)=\arg\min_{\alpha>0} f(x^k+\alpha d^k)$$

### 前提：单峰函数
如果 $\phi(\alpha)$ 是**单峰**的（先降后升，只有一个谷），就能用区间收缩法稳稳找到谷底。

### 常用一维方法
| 方法 | 原理 | 特点 |
|---|---|---|
| **黄金分割（0.618）** | 按黄金比例切区间，每次扔掉一段 | 只需函数值，不用导数，最常用 |
| **Fibonacci 法** | 类似黄金分割，用 Fib 数列定切点 | 固定次数下最优 |
| **二分法** | 用导数符号定位零点 $\phi'(\alpha)=0$ | 需要导数 |
| **一维牛顿 / 割线** | 用切线/割线逼近 $\phi'=0$ | 快但需导数、可能不稳 |
| **二次插值** | 用三点拟合抛物线，取顶点 | 收敛快 |

### 坑：精确线搜索其实"不划算"
真把每一维都解到精确，代价大且没必要 —— 我们只是想要"足够好"的一步，不是要把这条线上的最优算到小数点后十位。所以实战中更常用**非精确线搜索**。

## 3. 非精确线搜索：够用就好（实战主力）

### 思路
不追求最优 $\alpha$，只要这一步满足两个朴素要求就接受：
1. **函数确实降得够多**（别白走）；
2. **步子别太小**（别原地磨蹭）。

把这两点写成准则，就有了下面几个。

### 记号统一：一元函数 $\phi(\alpha)$

为了把下面所有准则写清楚，统一记
$$\phi(\alpha)=f(x^k+\alpha d^k),\qquad \phi(0)=f(x^k),\qquad \phi'(\alpha)=\nabla f(x^k+\alpha d^k)^Td^k.$$
特别地 $\phi'(0)=\nabla f(x^k)^Td^k<0$（因为 $d^k$ 是下降方向）。所有准则都是在比较 $\phi(\alpha)$ 与几条过点 $(0,\phi(0))$、斜率不同的**参考直线**。

### 3.1 Armijo 准则（保证"降得够"）

$$\phi(\alpha)\le \phi(0)+c_1\,\alpha\,\phi'(0),\qquad c_1\in(0,\tfrac12)$$

即 $f(x^k+\alpha d^k)\le f(x^k)+c_1\,\alpha\,\nabla f(x^k)^Td^k$。

**翻译**：实际下降量，要不小于"按梯度预测的下降量"的 $c_1$ 倍。右边那条参考线斜率为 $c_1\phi'(0)<0$，要求 $\phi$ 落在它**下方**。

**计算方法（回溯法 backtracking，最常用）**：
> 取初始 $\alpha=\alpha_0$（如 1）、缩减因子 $\beta\in(0,1)$（如 0.5）、参数 $c_1$（如 $10^{-4}$）。
> 1. 若 $\phi(\alpha)\le\phi(0)+c_1\alpha\,\phi'(0)$ 成立 → **接受**，返回 $\alpha$；
> 2. 否则 $\alpha\leftarrow\beta\alpha$，回到 1。

每次只需算一个函数值 $\phi(\alpha)$，简单稳健，[牛顿法代码](09_牛顿类算法.md)里用的就是它。

> **只有 Armijo 还不够**：它不阻止步长过小（$\alpha\to0$ 也能满足），可能磨蹭。所以要再补一条"下界"，防止步子太小——这就分出了 Wolfe 和 Goldstein 两套做法。

### 3.2 Wolfe 准则（Armijo + 曲率条件）

在 Armijo（下面记作条件 (W1)）基础上加一条**曲率条件 (W2)**：

$$\text{(W1)}\quad \phi(\alpha)\le \phi(0)+c_1\,\alpha\,\phi'(0)$$
$$\text{(W2)}\quad \phi'(\alpha)\ge c_2\,\phi'(0),\qquad 0<c_1<c_2<1$$

**翻译 (W2)**：走到新点后，沿原方向的斜率 $\phi'(\alpha)$ 不能还像出发时那么陡（负得那么多），要"平缓"起来，说明你确实走够远、跨过了最陡的下坡段。常用 $c_1=10^{-4}$，$c_2=0.9$（牛顿/拟牛顿）或 $0.1$（共轭梯度）。

**强 Wolfe**：把 (W2) 收紧为 $|\phi'(\alpha)|\le c_2|\phi'(0)|$，同时防止斜率变得太正（步子过大冲过头）。

**计算方法（两阶段：bracketing + zoom）**：单纯回溯无法满足 (W2)（回溯只会越缩越小、斜率不一定回升），所以 Wolfe 的标准实现分两段——
> **① 扩张定界**：$\alpha$ 从小往大试，直到发现某个区间 $[\alpha_{lo},\alpha_{hi}]$ 内必含满足 Wolfe 的点（即出现"违反 (W1) 或 $\phi$ 回升"或"$\phi'(\alpha)\ge0$"的信号）。
> **② zoom 内插**：在该区间用二次/三次插值取试探点，根据 (W1)、(W2) 是否满足收缩区间，直到命中。

**为什么 Wolfe 这么重要**：(W2) 保证了**曲率条件 $(s^k)^Ty^k>0$**（$s^k=x^{k+1}-x^k$ 位移、$y^k=\nabla f^{k+1}-\nabla f^k$ 梯度变化）。这是[拟牛顿 BFGS](10_拟牛顿类算法.md) **保持正定**的命根子，所以拟牛顿必须配 Wolfe，光 Armijo 不行——记住这个搭配。

### 3.3 Goldstein 准则（★考点，重点掌握）

**思想**：和 Wolfe 一样要"防止步长过大、也防止过小"，但 Goldstein **不用新点的导数**，只用两条参考直线把 $\phi(\alpha)$ 上下夹住。取参数 $c\in(0,\tfrac12)$（很多教材记作 $\rho$）：

$$\boxed{\ \phi(0)+(1-c)\,\alpha\,\phi'(0)\ \le\ \phi(\alpha)\ \le\ \phi(0)+c\,\alpha\,\phi'(0)\ }$$

写回原变量：
$$f(x^k)+(1-c)\,\alpha\,\nabla f_k^Td^k\ \le\ f(x^k+\alpha d^k)\ \le\ f(x^k)+c\,\alpha\,\nabla f_k^Td^k.$$

**两条线的几何**（关键，画个图就懂）：两条线都过 $(0,\phi(0))$，斜率分别是 $c\,\phi'(0)$ 和 $(1-c)\phi'(0)$，都是负的。因为 $c<\tfrac12$，有 $1-c>c$，所以 $(1-c)\phi'(0)$ **更陡（更负）**，是**下面那条线**；$c\,\phi'(0)$ 较缓，是**上面那条线**。

- **上界（= Armijo 条件）**：要求 $\phi$ 在上线之下 → 保证**下降得够多**，挡住**步长过大**。
- **下界**：要求 $\phi$ 在下线之上 → 挡住**步长过小**（$\alpha$ 太小时 $\phi(\alpha)\approx\phi(0)+\alpha\phi'(0)$ 比下线还低，被排除）。

可接受的 $\alpha$ 就是让 $\phi(\alpha)$ **夹在上下两条线之间**的那段区间。

**和 Wolfe 的区别（常考对比）**：
| | 下界用什么 | 要不要新点梯度 | 缺点 |
|---|---|---|---|
| **Wolfe** | 曲率条件 $\phi'(\alpha)\ge c_2\phi'(0)$ | **要**算 $\nabla f$ | 每次试探多算一个梯度 |
| **Goldstein** | 函数值下线 $\phi(\alpha)\ge\phi(0)+(1-c)\alpha\phi'(0)$ | **不要**，只算函数值 | 下线可能把真正的极小点 $\alpha^*$ 排除在外 |

> 一句话记忆：**Goldstein 用"两条线夹函数值"，Wolfe 用"一条线 + 斜率变缓"**。Goldstein 便宜（不用梯度），但可能误杀极小点；Wolfe 贵一点但能保住极小点和拟牛顿所需的曲率条件。

**计算方法（区间收缩 / 二分型）**：
> 维护区间 $[a,b]$，初始 $a=0,\ b=+\infty$（或一个大数），取试探 $\alpha>0$，参数 $c\in(0,\tfrac12)$。记上界条件 (G上)、下界条件 (G下)。
> 1. **若违反上界 (G上)**（$\phi(\alpha)$ 太大 → 步长过大）：令 $b\leftarrow\alpha$，缩小 $\alpha$（如 $\alpha\leftarrow\tfrac{a+b}{2}$）。
> 2. **若违反下界 (G下)**（$\phi(\alpha)$ 太小 → 步长过小）：令 $a\leftarrow\alpha$，放大 $\alpha$（$b<\infty$ 时取 $\tfrac{a+b}{2}$，否则 $\alpha\leftarrow2\alpha$）。
> 3. **两条都满足** → 接受，返回 $\alpha$。

**算例（按这个流程手算一遍）**：设沿某方向得到 $\phi(\alpha)=\alpha^2-2\alpha$，则 $\phi(0)=0,\ \phi'(0)=-2<0$（合法下降方向），真正的极小点 $\alpha^*=1$。取 $c=0.25$。

- 上线：$\phi(0)+c\,\alpha\,\phi'(0)=0+0.25\alpha(-2)=-0.5\alpha$
- 下线：$\phi(0)+(1-c)\alpha\,\phi'(0)=0+0.75\alpha(-2)=-1.5\alpha$
- 可接受条件：$-1.5\alpha\le \alpha^2-2\alpha\le -0.5\alpha$
  - 上界：$\alpha^2-2\alpha\le-0.5\alpha\Rightarrow\alpha^2-1.5\alpha\le0\Rightarrow 0\le\alpha\le1.5$
  - 下界：$\alpha^2-2\alpha\ge-1.5\alpha\Rightarrow\alpha^2-0.5\alpha\ge0\Rightarrow\alpha\ge0.5$
  - **可接受区间 $\alpha\in[0.5,\ 1.5]$**，极小点 $\alpha^*=1$ 落在其中。

迭代过程：取初始 $\alpha=2$。检查上界 $\phi(2)=0$ 与上线值 $-0.5\cdot2=-1$，$0\le-1$ 不成立 → 违反上界（步太大），$b\leftarrow2$，$\alpha\leftarrow\tfrac{0+2}{2}=1$。再查 $\alpha=1$：$\phi(1)=-1$，上线 $-0.5$、下线 $-1.5$，有 $-1.5\le-1\le-0.5$ ✓ 两条都满足 → **接受 $\alpha=1$**。

### 3.4 非单调线搜索（Grippo、Zhang-Hager）
允许函数值**偶尔上升**（不强求每步都下降），只要相对最近几步的最大值有下降即可。好处：在狭窄弯曲的"山谷"里能走得更顺、避免被迫走极小步。[BB 方法](08_梯度类算法.md)常配它使用。

## 4. Zoutendijk 定理：线搜索为什么能保证收敛

这是把"每步用 Wolfe 线搜索"和"全局收敛"连起来的理论桥。

**结论（直觉版）**：只要每步用 Wolfe 准则，就有

$$\sum_k \cos^2\theta_k\,\|\nabla f(x^k)\|^2<\infty$$

其中 $\theta_k$ 是搜索方向 $d^k$ 与负梯度的夹角。

**怎么读它**：这个无穷和收敛，意味着 $\cos^2\theta_k\|\nabla f(x^k)\|^2\to0$。所以**只要方向别太"歪"**（$\cos\theta_k$ 不趋于 0，即方向始终和负梯度保持锐角、不退化），就必有 $\|\nabla f(x^k)\|\to0$ —— 算法收敛到驻点！

> **实战启示**：设计算法时，保证两点就有全局收敛：① 方向是"够正"的下降方向（$\cos\theta_k$ 有下界）；② 步长满足 Wolfe。这就是后面各章证明收敛性的统一套路。

## 小结与下一步

- 迭代框架 $x^{k+1}=x^k+\alpha_k d^k$：方向 + 步长，本章管步长。
- 收敛速率：次线性 < 线性 < 超线性 < 二次；牛顿二次最快但每步贵。
- 精确线搜索（黄金分割等）求最优步长，但通常不划算。
- **非精确线搜索是主力**（统一用 $\phi(\alpha)=f(x^k+\alpha d^k)$ 看）：
  - **Armijo**：$\phi(\alpha)\le\phi(0)+c_1\alpha\phi'(0)$，回溯法实现（降得够）。
  - **Wolfe**：Armijo + 曲率 $\phi'(\alpha)\ge c_2\phi'(0)$，bracketing+zoom 实现，拟牛顿必备。
  - **Goldstein（考点）**：两条线 $\phi(0)+(1-c)\alpha\phi'(0)\le\phi(\alpha)\le\phi(0)+c\alpha\phi'(0)$（$c\in(0,\tfrac12)$）夹住，区间二分实现，不用新点梯度但可能误杀极小点。
  - **非单调**：容忍偶尔上升。
- Zoutendijk：方向够正 + Wolfe 步长 ⟹ 全局收敛。
