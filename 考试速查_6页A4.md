# 最优化 · 考试速查（3 张 A4 / 6 页）

> 用法：每页一个题型块，按「识别 → 步骤 → 公式 → 易错」照着做。打印时建议小四号、窄边距、双栏。★ = 模拟题已考或必考。

---

# ── 第 1 页：凸性判定 + 问题分类 ──

## 【题型 A】判断函数凸/凹 ★

**一元 $f(x)$**：求 $f''(x)$，看符号。
- $f''\ge0$ 全程 → **凸**；$f''\le0$ 全程 → **凹**；变号 → **都不是**（找拐点）。
- ⚠ 必须**整个定义域**不变号。Sigmoid $\frac{1}{1+e^{-x}}$：$f''=\sigma(1-\sigma)(1-2\sigma)$，$x=0$ 变号 → 既不凸也不凹。

**多元 $f(\boldsymbol x)$**：求 Hesse $\nabla^2 f$，判半正定。
- $\nabla^2 f\succeq0$ 全程 → 凸；$\succ0$ → 严格凸；$\preceq0$ → 凹；不定 → 都不是。
- **判半正定（任选一）**：① 全部特征值 $\ge0$；② 全部**顺序主子式** $\ge0$；③ $v^T\nabla^2 f\,v\ge0\ \forall v$；④ 凑成 $\frac{c}{(\cdot)}vv^T$（$c>0$ 则半正定）。
- 二阶判据不行时（如带绝对值/不可导）用**定义**：$f(\theta x{+}(1{-}\theta)y)\le\theta f(x){+}(1{-}\theta)f(y)$。

**例**（quadratic-over-linear）$f=\frac{x_1^2}{x_2},x_2>0$：$\nabla^2 f=\frac{2}{x_2^3}\begin{bmatrix}x_2\\-x_1\end{bmatrix}\begin{bmatrix}x_2&-x_1\end{bmatrix}=\frac{2}{x_2^3}vv^T\succeq0$ → 凸。

**保凸运算**（快速判复合函数）：非负加权和、与仿射复合 $f(Ax+b)$、逐点 **max**/上确界、$\max\{0,\cdot\}$。⚠ 逐点 min、并集**不**保凸。

**常用凸函数库**：线性/仿射、$\|x\|$（任意范数）、$x^TQx\ (Q\succeq0)$、$e^x$、$-\ln x$、$x\ln x$、$\max_i x_i$、$\frac{x^2}{y}(y>0)$、$\|Ax-b\|^2+\mu\|x\|_1$。

## 【知识点 B】凸优化问题 & 强凸/L-光滑
- 凸优化 = 凸目标 + 凸不等式约束 + **仿射**等式约束（⚠ 等式必须线性）。局部最优 = 全局最优。
- 层级 LP⊂QP⊂QCQP⊂SOCP⊂SDP。
- **$m$-强凸**：$\nabla^2 f\succeq mI$（解唯一）；**$L$-光滑**：$\nabla^2 f\preceq LI$，二次上界 $f(y)\le f(x)+\nabla f(x)^T(y-x)+\frac L2\|y-x\|^2$。条件数 $\kappa=L/m$（决定算法快慢）。

# ── 第 2 页：无约束最优性 + 最小二乘 ──

## 【题型 C】求驻点并分类（鞍点/极小/极大/全局）★

**步骤模板**：
1. **求梯度令零** $\nabla f=0$ → 解出所有驻点（逐个分量方程联立）。
2. **求 Hesse** $\nabla^2 f$，代入每个驻点。
3. **定号判类型**：

| $\nabla f$ | $\nabla^2 f$ | 结论 |
|---|---|---|
| $=0$ | 正定 $\succ0$ | 严格局部**极小** |
| $=0$ | 负定 $\prec0$ | 严格局部**极大** |
| $=0$ | 不定 | **鞍点** |
| $=0$ | 半正/半负定(含0) | 不定，需配方/高阶 |

4. **判全局**：用**配方法**把 $f$ 写成平方和 + 常数，读下界；或若 $f$ 凸则局部即全局。

**例** $f=x_1^4-2x_1^2+x_2^2+2x_2x_3+2x_3^2$：驻点 $(0,0,0)$[鞍]、$(\pm1,0,0)$[严格全局min]。配方 $f=(x_1^2-1)^2-1+(x_2+x_3)^2+x_3^2\ge-1$。
⚠ 全局最小可能**不唯一**（本例两点都取 $-1$）。⚠ Hesse 分块时，先判小块定号再看剩下对角元。

## 【题型 D】线性最小二乘
$\min\frac12\|Ax-b\|^2$ → 求梯度令零 → **正规方程** $\boxed{A^TAx=A^Tb}$，$A$ 列满秩则 $x=(A^TA)^{-1}A^Tb$。
- 病态（$A^TA$ 条件数大）→ **岭/Tikhonov**：$\min\frac12\|Ax-b\|^2+\frac\lambda2\|x\|^2$ → $(A^TA+\lambda I)x=A^Tb$。

## 【知识点 E】存在性
连续 + 紧集 → 最值存在（Weierstrass）。无界域：**强制**（$\|x\|\to\infty,f\to\infty$）或**强凸** → 最小存在（强凸还唯一）。

# ── 第 3 页：线搜索 + 梯度类算法 ──

## 【题型 F】固定步长梯度下降收敛性 ★
迭代 $x_{k+1}=x_k-\alpha\nabla f$。一维 $f=\frac12 ax^2$（$a=L$）：$x_{k+1}=(1-\alpha a)x_k$。
- **收敛区间**：$|1-\alpha a|<1\Rightarrow \boxed{0<\alpha<2/a\ (=2/L)}$。
- **最快步长**：$|1-\alpha a|$ 最小 → $\boxed{\alpha=1/a=1/L}$（一步到位）。
- $\alpha=2/L$ 临界震荡，$\alpha>2/L$ 发散。

## 【题型 G】最速下降法（精确步长）手算
方向 $d^k=-\nabla f(x^k)$；二次型 $f=\frac12 x^TAx-b^Tx$ 精确步长：
$$\boxed{\alpha_k=\frac{g_k^Tg_k}{g_k^TAg_k}},\quad g_k=Ax^k-b,\quad x^{k+1}=x^k-\alpha_k g_k$$
- 相邻方向正交 → **锯齿**；收敛率 $\left(\frac{\kappa-1}{\kappa+1}\right)^2$，$\kappa=\lambda_{\max}/\lambda_{\min}$（病态极慢）。

## 【知识点 H】线搜索准则
- **Armijo**（够降）：$f(x{+}\alpha d)\le f(x)+c_1\alpha\nabla f^Td,\ c_1\in(0,\tfrac12]$。回溯：$\alpha\leftarrow\beta\alpha$ 直到满足。
- **Wolfe** = Armijo + 曲率 $\nabla f(x{+}\alpha d)^Td\ge c_2\nabla f^Td$（$c_2\in(c_1,1)$）。**拟牛顿必配 Wolfe**（保 $s^Ty>0$）。
- **黄金分割**（0.618，单峰，只需函数值）：每次留 0.618 区间，扔一段。
- 收敛速率：次线性 $O(1/k)$ < 线性 $c^k$ < 超线性 < 二次 $e_{k+1}\le ce_k^2$。

## 【知识点 I】梯度法收敛总表（步长 $1/L$）
| 假设 | 度量 | 速率 | 复杂度 |
|---|---|---|---|
| 非凸+光滑 | $\min\|\nabla f\|^2$ | $O(1/k)$ | $O(1/\varepsilon^2)$ |
| 凸+光滑 | $f-f^*$ | $O(1/k)$ | $O(1/\varepsilon)$ |
| 强凸+光滑 | $\|x^k-x^*\|$ | 线性$(\frac{\kappa-1}{\kappa+1})^k$ | $O(\log\frac1\varepsilon)$ |
- **BB 步长**：$\alpha_{BB1}=\frac{s^Ty}{y^Ty},\ \alpha_{BB2}=\frac{s^Ts}{s^Ty}$，$s=x^k{-}x^{k-1},y=g_k{-}g_{k-1}$（非单调，加速）。

# ── 第 4 页：牛顿 + 拟牛顿(BFGS) + 共轭梯度 ──

## 【题型 J】牛顿法手算
牛顿方程 $\boxed{\nabla^2 f(x^k)\,d^k=-\nabla f(x^k)}$，$x^{k+1}=x^k+d^k$。
- 二次型**一步到最优** $x^*=Q^{-1}b$。二次收敛但需 Hesse 正定。
- 阻尼牛顿 = 牛顿方向 + 线搜索；修正牛顿 = $\nabla^2 f+\tau I$ 掰正。

## 【题型 K】BFGS 拟牛顿手算 ★（高频大题）
记 $g_k=\nabla f(x^k)=Qx^k-b$，$s^k=x^{k+1}{-}x^k$，$y^k=g_{k+1}{-}g_k$，$\rho=\frac{1}{(s^k)^Ty^k}$。

**每轮步骤**：
1. 方向 $\boldsymbol d^k=-H_k g_k$。
2. 步长（二次型精确）$\alpha_k=-\dfrac{g_k^T d^k}{(d^k)^T Q d^k}$。
3. 更新点 $x^{k+1}=x^k+\alpha_k d^k$，算 $g_{k+1},s^k,y^k$。
4. 更新 $H$（**逆 Hesse 近似，记这个**）：
$$\boxed{H_{k+1}=(I-\rho s^k y^{kT})H_k(I-\rho y^k s^{kT})+\rho s^k s^{kT}}$$
5. $g_{k+1}=0$ 则停，$x^*=x^{k+1}$。

**模拟题答案**（$Q=\begin{bmatrix}5&-3\\-3&2\end{bmatrix},b=(0,1)^T,H_0=I,x^0=0$）：
$\alpha_0=\tfrac12\to x^1=(0,\tfrac12)$；$H_1=\begin{bmatrix}1&\tfrac32\\\tfrac32&\tfrac{11}4\end{bmatrix}$；$\alpha_1=2\to x^2=(3,5)$，$g_2=0$ 停。$f_{\min}=7.5$，且 $H_2=Q^{-1}=\begin{bmatrix}2&3\\3&5\end{bmatrix}$。
⚠ $n$ 维正定二次 + $H_0=I$ → 最多 **$n$ 步**收敛、$H_n=Q^{-1}$。⚠ 别忘算 $f_{\min}$（代回原式）。

- **割线方程** $B^{k+1}s^k=y^k$（或 $H^{k+1}y^k=s^k$）；**曲率条件** $s^Ty>0$ 保正定。
- **L-BFGS**：只存最近 $m$ 组 $(s,y)$，$O(mn)$ 内存，大规模用。

## 【题型 L】共轭梯度（线性，解 $Ax=b$）
共轭 $u^TAv=0$。残差 $r_k=b-Ax_k$，方向 $v_{k+1}=r_k-s_k v_k$（$s_k$ 由 $v_k^TAv_{k+1}=0$ 定）。
- $\{r_k\}$ 两两正交、$\{v_k\}$ 两两共轭 → **不超过 $n$ 步**精确收敛。
- 非线性 CG：$d_{k+1}=-g_k-\beta_k d_k$，超线性。速度/代价：梯度法 < CG < 牛顿。

# ── 第 5 页：对偶 + KKT（约束最优性）──

## 【知识点 M】Lagrange 对偶
$L(x,\lambda,\nu)=f+\sum\lambda_i c_i^{\mathcal I}+\sum\nu_j c_j^{\mathcal E}$（$\lambda\ge0$）。对偶函数 $g=\inf_x L$（**恒凹**）。
- 弱对偶 $d^*\le p^*$（恒成立）；强对偶 $d^*=p^*$ 需**凸 + Slater**（存在严格可行点）。
- 乘子 = 影子价格（约束放松 1 单位，最优值改善量）。

## 【题型 N】KKT / 拉格朗日乘子法求约束最优 ★

**KKT 四条件**（需约束品性 LICQ）：
$$①\ \nabla f+\textstyle\sum\lambda_i\nabla c_i^{\mathcal I}+\sum\nu_j\nabla c_j^{\mathcal E}=0\quad ②\ \text{原始可行}\quad ③\ \lambda_i\ge0\quad ④\ \lambda_i c_i=0$$

**纯等式约束步骤**（如 $\min f$ s.t. $c(x)=0$）：
1. $L=f+\lambda c$（或 $\nu$）。
2. 联立 $\nabla_x L=0$ 与 $c(x)=0$ 解出所有候选 $(x,\lambda)$。
3. **比较各候选点 $f$ 值**取 min（或 max）。⚠ KKT 给的是所有极值候选，min/max 都在里面！

**例** $\min x_1x_2$ s.t. $x_1^2+x_2^2=1$：$x_2+2\lambda x_1=0,x_1+2\lambda x_2=0\Rightarrow\lambda=\pm\tfrac12$。
$\lambda=\tfrac12$：$x_2=-x_1$，$(\tfrac1{\sqrt2},-\tfrac1{\sqrt2})$ 等，$f=-\tfrac12$（**min**）；$\lambda=-\tfrac12$：$f=+\tfrac12$（max）。

**含不等式约束步骤**（猜起作用集）：
1. 列 KKT 四条。2. 对每条不等式分两种情形：**起作用**（$c_i=0$）或**不起作用**（$\lambda_i=0$）。
3. 每种组合解方程组。4. **验证** $\lambda_i\ge0$ 且原始可行，淘汰不合格者。
5. 凸问题：满足 KKT 即全局最优。

**EQP**（等式约束二次规划）KKT 线性系统：$\begin{bmatrix}G&A^T\\A&0\end{bmatrix}\begin{bmatrix}x\\\nu\end{bmatrix}=\begin{bmatrix}-c\\b\end{bmatrix}$。

# ── 第 6 页：罚函数 + 增广拉格朗日 + 易错清单 ──

## 【题型 O】增广拉格朗日法 (ALM) 手算 ★

**等式约束** $\min f$ s.t. $c(x)=0$：
$$\boxed{L_\sigma(x,\lambda)=f(x)+\lambda\,c(x)+\frac\sigma2 c(x)^2}$$

**步骤模板**：
1. **写 $L_\sigma$**（目标 + $\lambda c$ + $\frac\sigma2 c^2$）。
2. **子问题极小**：$\nabla_x L_\sigma=0$ 解出 $x^{k+1}$（含 $\lambda$ 表达式）。
3. **乘子递推**：$\boxed{\lambda^{k+1}=\lambda^k+\sigma\,c(x^{k+1})}$，代入 $x^{k+1}$ 化成 $\lambda$ 的线性递推。
4. **求极限**：解不动点 $\lambda^*=\lambda^*$ 方程（递推 $|$系数$|<1$ 必收敛）。
5. **代回**得最优 $x^*$。

**例** $\min x^2+y^2$ s.t. $x+y-2=0$，$\sigma=1$，$\lambda^0=0$：
- $L=x^2+y^2+\lambda(x+y-2)+\tfrac12(x+y-2)^2$；
- $\nabla_x L=0$：$x=y=\frac{2-\lambda}{4}$；
- 递推 $\lambda^{k+1}=\tfrac12\lambda^k-1$ → 极限 $\lambda^*=-2$；
- 代回 $x=y=\frac{2-(-2)}{4}=1$，$f=2$。
⚠ ALM 优势：$\sigma$ 固定有限即可精确收敛（靠乘子更新），不必 $\sigma\to\infty$。
- **不等式**乘子更新：$\mu^{k+1}=\max\{\mu^k+\sigma c,0\}$。

## 【知识点 P】罚函数法
- **二次罚（外点）** $P=f+\frac\sigma2\sum c_i^2$，需 $\sigma\to\infty$（病态）。⚠ $\sigma$ 太小可能无下界。
- **对数罚（内点）** $P=f-\sigma\sum\ln(-c_i)$，需可行初值、$\sigma\to0$。
- **$\ell_1$ 精确罚** $P=f+\sigma\sum|c_i|$，有限 $\sigma>\|\lambda^*\|_\infty$ 即精确，但非光滑。

## ★★ 通用易错清单（交卷前自查）★★
1. 凸性：$f''$/Hesse 要**全程**不变号；半正定别误判成正定。
2. 驻点题：**别漏驻点**（$x(x^2-1)=0$ 有 3 根）；全局最小可能不唯一；记得判是否全局。
3. 梯度/最速下降：步长范围 $(0,2/L)$，最优 $1/L$。
4. BFGS：用 $H$（逆）公式；二次型步长是精确公式；**最后代回算 $f_{\min}$**；验证 $H_n=Q^{-1}$。
5. KKT/拉格朗日：解出**全部**候选，**比较 $f$ 值**定 min/max；不等式乘子 $\lambda\ge0$、等式乘子无号限制；互补松弛 $\lambda_i c_i=0$。
6. ALM：乘子递推 $\lambda^{k+1}=\lambda^k+\sigma c$；极限解不动点；$\sigma$ 不必趋于 ∞。
7. 收敛速率：线性/超线性/二次会判断；条件数 $\kappa$ 大 = 慢。
8. 矩阵运算：$\nabla(\frac12 x^TQx-b^Tx)=Qx-b$，$\nabla^2=Q$（$Q$ 须对称，不对称先对称化 $\frac{Q+Q^T}{2}$）。

**速记口诀**：判凸看二阶｜驻点令梯零、定号分类型、配方判全局｜步长 $1/L$ 最快、$2/L$ 是界｜BFGS 记 $H$ 公式、二次 $n$ 步到、补算 $f$｜约束写拉氏、解全部、比 $f$ 值｜ALM 更新乘子求极限、$\sigma$ 不用大。
